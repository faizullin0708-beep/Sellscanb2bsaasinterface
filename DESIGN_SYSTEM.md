## 🎬 Анимации

### Переходы страниц
- **Entry**: opacity 0→1, translateY 20px→0, duration 300ms
- **Exit**: opacity 1→0, translateY 0→-20px, duration 300ms

### Карточки
- **Entry**: opacity 0→1, translateY 20px→0, stagger delay 50-100ms
- **Hover**: translateY -4px, shadow увеличивается, duration 200ms

### Кнопки
- **Hover**: scale 1.02, duration 200ms
- **Active**: scale 0.98, duration 200ms

### Таблицы
- **Rows**: opacity 0→1, translateY 10px→0, stagger delay 30ms per row

### Список элементов
- **Entry**: opacity 0→1, translateX -20px→0, stagger delay 50ms

## 🎯 Принципы дизайна

1. **Много воздуха** - generous whitespace
2. **Четкая иерархия** - clear typography scale
3. **Акцент на данных** - tables and charts are primary
4. **Минимум визуального шума** - clean and focused
5. **Профессиональность** - Stripe/Linear/Vercel level
6. **Быстрая читаемость** - scan-friendly layouts
